Keyboard events

The keypress event captures the printable keys such as letters, symbols and enter key. The key will be captured when it has been pressed and not released yet.

The keydown event captures the printable and non-printable keys such as letters, symbols, modifier keys (Shift, Ctrl) etc. and captured when the keys has been pressed.

Better to use onkeypress where only text fields, form editors etc. data required and use onkeydown when need all character such as in the game controls, shortcuts etc.
